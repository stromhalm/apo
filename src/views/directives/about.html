About This Project

APO is a web interface for APT, a powerful software for Petri net analysis and synthesis developed at the Carl von Ossietzky University in Oldenburg, Germany. APO is developed by Adrian Jagusch and was part of his bachelor thesis.

To navigate in big nets using the browsers built in zoom functions (usually Cmd+'+', Cmd+'-' and Cmd+0) may help.
